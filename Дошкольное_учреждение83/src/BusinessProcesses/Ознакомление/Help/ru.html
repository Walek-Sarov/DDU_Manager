Этот бизнес-процесс используется для передачи коллегам каких-либо материалов (документы, файлы, другие данные системы) для обязательного ознакомления.

Этот бизнес-процесс можно создавать непосредственно из карточки, например, внутреннего документа, при помощи меню "Создать на основании".

При старте этого бизнес-процесса все исполнители по списку автоматически получат задачу "Ознакомиться".

Для просмотра текущего состояния бизнес-процесса (списка выполненных и не выполненных задач) используйте команду "Задачи" в панели навигации в карточке бизнес-процесса.

См. также: Бизнес-процессы, Задачи.
