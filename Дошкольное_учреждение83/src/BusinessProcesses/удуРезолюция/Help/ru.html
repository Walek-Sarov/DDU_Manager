Этот бизнес-процесс используется для автоматизации исполнения распоряжений руководителя одним или несколькими исполнителями с одновременным контролем.

Если указано несколько исполнителей, то одного из них можно указать ответственным исполнителем. При старте этого бизнес-процесса соответствующие задачи на исполнение автоматически получат все исполнители по списку и контролер. Из задачи ответственного исполнителя имеется возможность просмотреть задачи остальных исполнителей и при необходимости вернуть исполнителю задачу на доработку.

Этот бизнес-процесс можно создавать непосредственно из карточки, например, входящего документа, при помощи меню "Создать на основании". Также этот бизнес-процесс может быть создан из задачи "Обработать резолюцию" при помощи меню "Создать на основании".

Для просмотра текущего состояния бизнес-процесса (списка выполненных и не выполненных задач) используйте команду "Задачи" в панели навигации в карточке бизнес-процесса.

Если предметом исполнения является входящий или внутренний документ, то при старте этого бизнес-процесса документу автоматически будет установлено состояние "На исполнении" и при успешном завершении бизнес-процесса - состояние "Исполнен".
