Этот бизнес-процесс используется для передачи исходящего или внутреннего документа на утверждение руководителю. 

При старте бизнес-процесса руководителю поступает задача "Утвердить", содержащая ссылку на документ. После того как руководитель рассмотрит документ и отметит в задаче результат утверждения, автору поступает задача "Ознакомиться с результатами утверждения". В случае отрицательного результата из этой задачи автор может отправить документ на повторное утверждение. В случае положительного результата из этой задачи автор может направить документ на регистрацию при помощи меню "Создать на основании".

Этот бизнес-процесс можно создавать непосредственно из карточки, например, исходящего документа, при помощи меню "Создать на основании".

При выборе утверждающего можно указать не только конкретного пользователя, но и роль.

Для просмотра текущего состояния бизнес-процесса (списка выполненных и не выполненных задач) используйте команду "Задачи" в панели навигации в карточке бизнес-процесса.

Если предметом исполнения является исходящий или внутренний документ, то при старте этого бизнес-процесса документу автоматически будет установлено состояние "На утверждении" и при успешном завершении бизнес-процесса - состояние "Утвержден".
