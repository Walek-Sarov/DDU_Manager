Журнад документов "Книга кадровых документов"

 

Журнал содержит перечень документов по приему, перемещению и увольнению из организаций.

Просмотр списка документов можно осуществлять по организации, по сотруднику, в оперделенном диапозоне дат или без ограничения по кнопке "Установить интервал дат для просмотра".
