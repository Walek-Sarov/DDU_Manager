Отчет по заявкам на зачисление в группу

Отчет составляется на основании документов "Заявка на зачисление ребенка в ДОУ".

Предназначен для показа  очереди в ДОУ, отражения информации по заявкам на зачисление.

Отчет может формироваться по всем заявкам или по заявкам в одну конкретную группу (реквизит "Группа для зачисления").

Также есть три варианта формирования отчета по состояниям заявки:

  • Заявки, по которым произошло зачисление в группу;
  • Заявки, по которым планируется зачисление в группу;
  • Заявки, по которым отказано в зачислении в группу.

 

Если в настройках программы стоит отметка об использовании возрастного признака при комплектовании групп, тогда в отчете также можно установить отбор групп по возрастному признаку.
