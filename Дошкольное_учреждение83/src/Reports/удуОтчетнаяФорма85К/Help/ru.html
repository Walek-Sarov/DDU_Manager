Отчет "Сведения о деятельности дошкольного образовательного учреждения (форма 85-К)"

Сведения по форме N 85-К представляют образовательные учреждения, реализующие общеобразовательные программы дошкольного образования всех форм собственности и ведомственной принадлежности.

Отчет по форме составляется на конец отчетного года. Показатели, исчисляемые на определенную дату (например, численность детей, наличие групп и мест, штаты), заполняются по состоянию на 31 декабря соответствующего года.

Все показатели, приведенные в форме, должны соответствовать данным первичной учетной документации, имеющейся в образовательном учреждении. При заполнении формы должна быть обеспечена полнота заполнения и достоверность содержащихся в ней статистических данных.

 

Для формирования отчета необходимо указать учреждение, по которому будет составляться отчет и отчетный период (год). Отчет имеет 5 разделов:

Раздел 1. Общие сведения об учреждении:

  1.1.   Тип и вид учреждения;

  1.2.   Организационная структура учреждения;

Раздел 2. Сведения о численности детей:

  2.1.    Распределение детей по группам;

  2.2.    Распределение детей по возрасту;

  2.3.    Посещаемость учреждения;

  2.4.    Организация летнего отдыха детей; 

  2.5.    Число случаев заболевания детей;

  2.6.    Язык обучения и воспитания детей;

  2.7.    Группы кратковременного пребывания детей;

  2.8.    Платные дополнительные образовательные услуги;

  2.9.    Число кружков, секций и численность обучающихся в них;

Раздел 3. Сведения о персонале учреждения:

  3.1. Распределение персонала по уровню образования и полу;

  3.2. Распределение административного и педагогического персонала по возрасту;

  3.3. Распределение административного и педагогического персонала по стажу работы;

  3.4. Штаты дошкольного образовательного учреждения;

Раздел 4. Материально-техническая база учреждения:

  4.1. Площадь помещений дошкольного образовательного учреждения;

  4.2. Техническое состояние зданий дошкольного образовательного учреждения. Наличие компьютеров.

Раздел 5. Финансово-экономическая деятельность учреждения

  5.1. Распределение объема средств учреждения по источникам их получения;

  5.2. Расходы учреждения;
