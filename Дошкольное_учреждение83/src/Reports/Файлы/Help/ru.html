Для формирования отчета нажмите кнопку "Сформировать".

При необходимости установите параметры отчета:

Автор Будут отобраны только документы по указанному автору
Автор версии Будут отобраны только такие документы, автором текущей версии которых является указанный пользователь
Редактирует Будут отобраны документы, занятые для редактирования указанным пользователем
Период Будут отобраны только такие документы созданные за указанный период

 

Для детальной настройки отчета используйте команду "Все действия - Изменить вариант".
