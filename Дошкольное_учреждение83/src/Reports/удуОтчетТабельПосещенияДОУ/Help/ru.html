Отчет "Табель посещения ДОУ"

Отчет составляется на основании документов "Табель посещаемости ДОУ".

Перед формированием отчета необходимо указать группу, по которой будет формироваться табель и месяц, за который формировать.

Отметки в ячейках табеля:

  • "-" информация отсутствует;
  • Оу - ребенок отсутствовал по уважительно причине;
  • Он - отсутствовал по неуважительной причине;
  • П - ребенок присутсвовал на занятиях в ДОУ.

 

Также можно указать использовать ли цветовую окраску при формировании отчета, если отметка установлена, то для разных ячеек отчета будут применяться разные цвета, а на закладке "Оформление отчета" (становится доступной только если установлена отметка) можно настроить цветовую палитру для ячеек.
