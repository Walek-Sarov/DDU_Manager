Настройки работы с файлами

Настройка Описание
Действие по двойному щелчку мыши Указывает, какое действие выполнять при нажатии на клавишу Enter или двойному щелчку левой кнопкой мыши на файле - открыть файл или открыть его карточку.
Выбор режима открытия файла при двойном щелчке мыши Если установлена, то перед открытием файла откроется окно для выбора режима открытия - "просмотр" или "редактирование". 
Показывать подсказки при редактировании файлов

Если установлена, то при работе через веб клиент система будет подсказывать о том, как удобнее открыть файл для просмотре или редактировании и вернуть его в информационную базу.

Показывать колонку "Размер" в списках файлов Если включено, в форме хранилища документов и в списке редактируемых файлов показывается колонка Размер (в Кб).
Сравнивать версии файлов DOC, RTF, TXT при помощи

Выбор одного из двух возможных вариантов сравнения файлов указанного формата - "Microsoft Office Word" или "OpenOffice.org Writer".

Настройка рабочего каталога Нажатие на эту кнопку откроет окно управления рабочим каталогом.
Установить расширение работы с файлами на клиенте Устанавливает дополнительную компоненту для работы с файлами при подключении к информационной базе через веб-браузер.
Настройка сканирования Нажатие на эту кнопку откроет форму настройки ввода документов и файлов со сканера.

 

Работа с документами

Настройка Описание
Вид входящего документа Будет использоваться как вид по умолчанию при создании новых входящих документов.
Вид исходящего документа Будет использоваться как вид по умолчанию при создании новых исходящих документов.
Вид внутреннего документа Будет использоваться как вид по умолчанию при создании новых внутренних документов.
Основной способ получения Будет использоваться как способ получения по умолчанию при создании новых входящих документов.
Основной способ отправки Будет использоваться как способ отправки по умолчанию при регистрации исходящих документов.
Показывать предупреждение
при регистрации		 Если флажок установлен, то при регистрации документов будет выдаваться предупреждение о том, что после регистрации поля документов станут недоступны для редактирования.

 

Настройки учета времени

Настройка Описание
Основной вид работ Указывает вид работ, который будет использоваться по умолчанию при оформлении ежедневных отчетов.
Способ указания времени

Устанавливает один из способов указания длительности работ в ежедневных отчетах:

  • Длительность - для каждой работы указывается ее длительность 
  • Время начала - для каждой работы указывается время ее начала, окончание работы совпадает с началом следующей работы
    • Добавлять работу в ежедневный отчет при выполнении задач Если флажок установлен, то при выполнении задач в соответствующий ежедневный отчет автоматически будет добавлена строка о выполнении данной задачи с указанием затраченного времени и вида работ.
