Настройка программы выполняется администратором (пользователем с ролью "Полные права").

Общие параметры

Настройка Описание
Заголовок системы Будет использоваться как стандартный заголовок программы
Использовать хранилище файлов

Если установлена, то пользователи смогут хранить и обрабатывать в системе обычные файлы, организовывая их в структуру папок.

Использовать бизнес-процессы и задачи Если установлена, то в системе появляется раздел "Задачи и бизнес-процессы" и становятся доступными команды работы с задачами и бизнес-процессам  в формах документов и файлов.
Использовать учет времени Если установлена, то в системе появляется раздел "Учет времени".
Использовать ограничение прав доступа

Если установлена, то включается механизм ограничения прав доступа на файлы и документы. Если не установлена, то все пользователи получают неограниченный доступ к любым данным системы.

Не рекомендуется использовать ограничение прав доступа в файловом варианте информационной базы, т.к. это может привести к существенному замедлению работы программы.

Использовать дополнительные реквизиты и сведения Если установлена, то у администратора появляется возможность настройки дополнительных реквизитов и сведений, например, для входящих, исходящих или внутренних документов
Использовать полнотекстовый поиск Если установлена, то пользователи смогут вызывать полнотекстовый поиск данных. Построение полнотекстового индекса будет выполняться автоматически в случае клиент-серверного варианта и вручную в случае файлового варианта. 
Использовать версионирование объектов Если установлена, то система будет автоматически сохранять историю изменения реквизитов объектов.

 

Данные предприятия

Настройка Описание
Наименование предприятия Официальное наименование Вашей организации, которое будет использоваться для печати делопроизводственных бланков, например, номенклатуры дел.
Руководитель предприятия Пользователь, ФИО которого будет использоваться для печати делопроизводственных бланков.
Руководитель службы ДОУ Пользователь, ФИО которого будет использоваться для печати делопроизводственных бланков.

 

Делопроизводство

Настройка Описание
Использовать номенклатуру дел Если установлена, то в системе появляются соответствующие справочники ("Номенклатура дел", "Дела (тома)") и реквизиты в формах входящих, исходящих и внутренних документов.
Использовать журнал передачи Если установлена, то в системе появляется возможность ведения и просмотра журнала передачи бумажных документов исполнителям. Также в карточках входящих, исходящих и внутренних документов появляются команды перехода к записям о передачи этих документов исполнителям.
Использовать состав документов Если установлена, то в формах входящих, исходящих и внутренних документов появляются поля для указания количества листов, приложений и листов в приложениях.
Использовать виды входящих документов Если установлена, то в карточке входящего документа появляется обязательное для заполнения поле "Вид".
Использовать виды исходящих документов Если установлена, то в карточке исходящего документа появляется обязательное для заполнения поле "Вид".
Использовать виды внутренних документов Если установлена, то в карточке внутреннего документа появляется обязательное для заполнения поле "Вид".
Использовать файлы у входящих документов Если установлена, то в карточке входящего документа появляется возможность присоединить файлы и работать с ними.
Использовать файлы у исходящих документов Если установлена, то в карточке исходящего документа появляется возможность присоединить файлы и работать с ними.
Использовать связи документов Если установлена, то в карточках всех документов (входящих, исходящих и внутренних) появляется возможность настройки перекрестных ссылок документов друг на друга.
Использовать резолюции в документах Если установлена, то в карточках все документов (входящих, исходящих и внутренних) появляется группы полей для ввода резолюции руководителя по документу.
Использовать учет по организациям Если установлена, то программа будет вести учет всех ключевых данных в разрезе нескольких организаций. Для этого в карточках документов, номенклатуры дел и шаблонов бизнес-процессов появляется поле "Организация". Перед началом работы в режиме ведения учета по организациям необходимо заполнить справочник Организации. 
Использовать грифы доступа Если установлена, то доступ к входящим и исходящим документам будет ограничен грифами доступа. В формах входящего и исходящего документов также появится обязательное для заполнения поле гриф.

 

Работа с файлами

Настройка Описание
Максимальный размер файла Система не разрешает добавление в хранилище файла, размер которого превышает указанный. По умолчанию установлен равным 50 Мб.
Извлекать тексты только на сервере Указывает, что серверу необходимо самостоятельно извлекать тексты из файлов, для построения полнотекстового индекса. При этом извлечение текста выполняется регламентным заданием автоматически.
Извлечение текстов работает только на серверах под управлением ОС Windows.
Тип хранения файлов Устанавливает один из двух возможных типов хранения файлов - в информационной базе или в томах на диске. При смене типа хранения не происходит переноса файлов из информационной базы в тома и обратно. Если выбран тип хранения "Тома на диске", то нужно настроить список томов хранения файлов.
Запретить загрузку файлов с расширением С помощью этого флажка можно запретить загружать в информационную базу файлы с указанными расширениями. Список запрещенных расширений по умолчанию - «COM EXE BAT CMD VBS VBE JS JSE WSF WSH SCR»). При необходимости можно добавить свои расширения, запрещенные для загрузки.
Настройка автоматической загрузки файлов Открывает диалог для настройки автоматической загрузки файлов, которая выполняется с помощью регламентного задания.

 

Бизнес-процессы и задачи

Настройка Описание
Разрешить изменение даты исполнения задач Если установлена, то в формах задач всех пользователей станет доступным для изменения поле даты исполнения
Использовать перенаправление задач Если установлена, то система разрешает исполнителям перенаправлять свои задачи другим исполнителям.
Использовать дату и время в сроках задач Если установлена, то в задачах и бизнес-процессах даты будут отображаться с точностью до времени.
Разрешить ручное изменение состояния документов Если установлена, то пользователи смогут вручную менять состояния входящих, исходящих или документов. Если не установлена, то состояния документов будут изменяться только бизнес-процессами автоматически.
