Текущее состояние документа показывает этап прохождения документа в системе документооборота предприятия.

Указание состояния выполняется в карточке документа в поле "Состояние". Поле показывается, если настройка "Использовать состояние документа" установлена.

Для просмотра истории изменения состояния документа следует открыть карточку документа и в панели навигации выполните команду "История состояния". Команда показывается, если настройка "Использовать состояние документа" установлена.

Изменение состояния может быть выполнено вручную (в карточке) или с помощью бизнес-процесса.

Например, типовая процедура прохождения входящего документа будет выглядеть следующим образом:

23.03.2010 На регистрации Фролова Е.М. (секретарь)
23.03.2010 Зарегистрирован Фролова Е.М. (секретарь)

 

Для изменения состояния документа вручную следует в карточке документа в поле "Состояние" с помощью кнопки выбора открыть выпадающий список и выбрать нужное состояние. После записи карточки документа в историю состояния добавляется запись о выполнении изменения состояния. При этом в колонке "Установил" будет указан пользователь, установивший данное состояние.

Изменение состояния документа выполняется автоматически при создании нового бизнес-процесса на основани данного документа.
