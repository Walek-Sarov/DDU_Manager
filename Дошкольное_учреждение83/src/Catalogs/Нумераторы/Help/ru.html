Справочник "Нумераторы" служит для настройки правил формирования регистрационных номеров у входящих, исходящих и внутренних документов.

Нумератор определяет формат регистрационного номера, который может сконструировать (собрать из готовых фрагментов) в карточке нумератора.

Каждый нумератор имеет свой собственный нарастающий регистрационный номер, который не зависит от других нумераторов.

Нумератор можно назначается видам документов. Один нумератор можно назначить сразу нескольким видам, например, для того чтобы поддерживать для них сквозную нумерацию.
