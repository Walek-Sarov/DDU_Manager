Справочник "Авторы"

Предназначен для хранения основной информации об авторах методических материалов.

Назначение реквизитов

Код - код справочника. Код присваивается автоматически при записи элемента справочника.

Фамилия - фамилия автора.

Имя - имя автора.

Отчество - отчество автора.

Инициалы - иницалы автора. Формируются автоматически при заполнении имени и отчества автора.
