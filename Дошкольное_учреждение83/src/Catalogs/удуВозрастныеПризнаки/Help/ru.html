Справочник «Возрастные признаки»

Предназначен для хранения списка возрастных признаков воспитательных групп детского учреждения. 

Справочник одноуровневый.

Назначение реквизитов

Код - код справочника. Код присваивается автоматически при записи элемента справочника.

Наименование - наименование возрастного признака. Например "Старшие группы".

 

Справочник используется только в случае, если в настройках программы стоит отметка "использовать возрастной признак при комплектовании". В этом случае элементы справочника будут использоваться в справочнике "Группы (воспитательные)".
