Справочник «Национальности»

Предназначен для хранения списка национальностей. 

Назначение реквизитов

Код - код справочника. Код присваивается автоматически при записи элемента справочника.

Наименование - наименование национальности.

 

Элементы справочника используются в справочниках "Личные дела детей" и "Родители, члены семьи". 

Использование справочника возможно только если в настройках программы стоит отметка о возможности использования в конфигурации персональных данных физических лиц.

Внимание!  В случае добаления в информационную базу персональных данных, касающихся рассовой, национальной принадлежности, политических взглядов, религиозных и философских убеждений, состояния здоровья, личной жизни, а также дополнительной информации о субъекте, класс информационной системы обработки персональных данных будет повышен до уровня К1 или К2.

В соответствии с ФЗ №152-ФЗ "О персональных данных" и подзаконными актами требования, предъявляемые по защите таких даннх, будут максимальными.              Для проведения мероприятий по приведению в соответствие информационных систем персональных данных классов К1 и К2 в соответствие с требованиями по защите необходимо обратится в специализированную орагнизацию, имеющую лицензию ФСТЭК на предоставление деятельности по защите конфиденциальной информации.
