Справочник "Роли автора"

Предназначен для хранения перечня ролей (степеней ответвенности) авторов методического материала. Примеры ролей: научный редактор, оформитель и т.п.

Назначение реквизитов

Код - код справочника. Код присваивается автоматически при записи элемента справочника.

Наименование - наименование причины выдачи.

Сокращение - сокращенное название роли. Используется при формировании представления методического материала.
