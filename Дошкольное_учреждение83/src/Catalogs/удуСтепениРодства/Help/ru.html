Справочник «Степени родства»

Справочник показывает список возможных степеней родства.

Элементы справочника используются при создании элементов справочников "Личные дела детей" и "Родители, члены семьи".

 

При создании нового элемента указываются следующие данные:

Код - код справочника. Код присваивается автоматически при записи элемента справочника.

Наименование - наименование степени родства, к примеру "Отец".

Наименование родства ребенка - кем является ребенок (мальчик и девочка) по отношению к человеку этой степени родства.

Например: Степень родства "Отец", тогда наименование родства ребенка для девочки будет "Дочь", для мальчика - "Сын".
