Справочник "Серии (Периодические издания)"

Предназначен для хранения перечня периодических изданий, используемых в работе дошкольного учреждения.

Назначение реквизитов

Код - код справочника. Код присваивается автоматически при записи элемента справочника.

Наименование - наименование серии (периодического издания).

Тип периодического  издания - тип серии (периодического издания): газета, журнал, дайджест, прочее.

Учредитель - наименование организации-учредителя периодического издания.

Количество страниц - количество страниц периодического издание (заполняется, если иметт постоянное значение для разных номеров периодического издания).

Иллюстрации - признак наличия иллюстраций в издании.

Первый номер - номер, с которого был начат выпуск периодического издания (как правило, указывается в описании издания, по умолчанию равен 1).

Последний номер - номер, которым должна быть завершена серия (периодическое издание). Может быть пустым.

Представление - представление серии (приодического издания). Соответствует билиографическому описанию серии. Формируется автоматически на основании реквизитов экземпляра справочника. Может быть скорректировано вручную.

Сведения относящиеся к заглавию - участвует в формировании представления серии.

Общее обозначение материала - наименование вида материала (текст, видеозапись, ноты и т.д.).

Авторы - перечень авторов данного периодического издания с указанием степени ответсвенности (роли).

См. также Учетная единица.
