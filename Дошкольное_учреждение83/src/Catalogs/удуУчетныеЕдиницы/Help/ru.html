Справочник "Учетные единицы"

Предназначен для хранения информации о методических материалах, используемых в дошкольном учреждении. Является основное единицей хранения информациии подсистемы "Методические материалы".

Назначение реквизитов

Код - код справочника. Код присваивается автоматически при записи элемента справочника.

Наименование - наименование методического материала.

Авторы - перечень авторов данной статьи с указанием степени ответственности (роли).

Автор представление - содержит текстовое представление списка авторов методического материала с учетом ролей авторов.

Вид материала - вид методического материала.

Год - год издания методического материала.

Издательство - издательство, опубликовавшее данный методический материал.

Иллюстрации - признак наличия иллюстраций.

Ключевые слова - ключевые слова для поска.

Количество страниц - общее количество страниц.

Место издания - город, в котором был издан методический материал.

Количество страниц - общее количество страниц.

Многотомник - ссылка на многотомное издание, в которое включен данный материал. При указании многотомника автоматически заполняются одноименные реквизиты учетной единицы методического материала.

Направление деятельности ДОУ - направление деятельности дошкольного учреждения, при работе по которому может быть использован методический материал.

Недоступен к выдаче - признак запрета выдачи данного методического материала на руки.

Номер периодического издания - порядковый номер издания в серии. Используется только для элементов справочника, у которых вид методического материала равен "Серия (периодическое издание)".

Номер тома - порядковый номер тома в многотомном издании.

Образы - ссылка на электронные образы материала, размещеные в справочнике Файлы подсистемы Документы и файлы.

Общее обозначение материала - наименование вида материала (текст, видеозапись, ноты и т.д.).

Представление - представление методического материала. Соответствует билиографическому описанию материала. Формируется автоматически на основании реквизитов экземпляра справочника. Может быть отредактировано вручную.

Примечание - представление статьи. Соответствует билиографическому описанию серии. Формируется автоматически на основании реквизитов экземпляра справочника. Может быть отредактировано вручную.

Сведения относящиеся к заглавию - участвует в формировании представления серии.

Серия - серия (периодическое издание), в котрое входит данный методический материал.

Учредитель - наименование организации-учредителя периодического издания.
