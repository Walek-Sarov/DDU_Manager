Данный классификатор служит для подбора профессий рабочих.

В форме классификатора содержится таблица профессий, сгруппированных по алфавиту. Некоторые наименования в таблице выделены цветом - это означает, что данные профессии уже были добавлены ранее в справочник.

Чтобы добавить с помощью данного классификатора новые элементы, следует нажать кнопку "Выбрать", в результате будут добавлены те профессии, которые были отмечены галочкой. Можно отметить нужные профессии по одной или отметить флажок напротив строки с буквой алфавита - тогда автоматически будут отмечены все профессии, начинающиеся с этой буквы.
