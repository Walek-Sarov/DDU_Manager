Данный классификатор служит для подбора должностей служащих.

В форме классификатора содержится таблица должностей, сгруппированных по алфавиту. Некоторые наименования в таблице выделены цветом - это означает, что данные должности уже были добавлены ранее и при новом добавлении повторно в справочник должностей не попадут.

Чтобы добавить с помощью данного классификатора новые должноти следует нажать кнопку "Выбрать", в результате будут добавлены те должности, которые были отмечены галочкой. Можно отметить нужные должности по одной или отметить флажок напротив строки с буквой алфавита - тогда автоматически будут отмечены все должности, начинающиеся с этой буквы.
