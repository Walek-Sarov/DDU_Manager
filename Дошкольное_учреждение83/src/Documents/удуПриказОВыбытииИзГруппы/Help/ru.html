Документ «Приказ о выбытии из группы»

Документ располагается в блоке "Приказы о зачислении, переводе, выбытии".

Документ оформляется в случае выбытия ребенка из ДОУ или в случае перехода в другую группу.

В документе присутствуют следующие реквизиты:

Номер - номер приказа. Номер присваивается автоматически при записи документа.

Дата - дата приказа о выбытии.

Дата фактического выбытия - указывается дата фактического выбытия ребенка из группы.

Учреждение - указывается учреждение, из группы которого выбывает ребенок. Заполняется автоматически.

Группа выбытия - из справочника "Группы (воспитательные)" выбирается группа, из которой выбывает ребенок.

Причина выбытия (по умолчанию) - можно указать причину выбытия, которая будет проставлена всем детям в талбличной части. Для автоматической подстановки в табличную часть необходимо выбрать причину из справочника "Причины выбытия из группы", заполнить табличную часть списком выбывающих детей и затем нажать кнопку "Заполнить".

Заполнение табличной части происходит по кнопке "Заполнить табличную часть". В открывшейся форме необходимо оставить галочки у имен тех детей, которые будут отчислены (переведны в другую группу) и нажать кнопку "Перенести позиции в документ". В приказ попадут только те дети, напротив которых стояли галочки.
