Документ «Приказ о зачислении ребенка в группу»

Документ располагается в блоке "Приказы о зачислении, переводе, выбытии".

Предназначен для формирования приказа о зачислении ребенка в группу ДОУ.

Документ может формироваться автоматически из документа "Комплектование групп", при этом в документе появится ссылка на документ основание.

Назначение реквизитов

№ приказа - номер приказа, присваивается автоматически при записи документа.

Дата приказа - дата создания приказа о зачислении.

Дата фактического зачисления - указывается дата фактического зачисления ребенка в группы.

Учреждение - указывается учреждение, в группу которого зачисляется ребенок. Заполняется автоматически.

Группа - из справочника "Группы (воспитательные)" выбирается группа, в которую зачисляется ребенок.

Примечание.
После выбора группы рядом появится информация о колличестве человек в группе и о ее вместимости. Если при записи документа количество детей окажется больше, чем вместимость группы, программа выдаст сообщение об ошибке и не позволит записать документ. 

Для отключения контроля вместимости группы необходимо установить отметку "Отключить контроль вместимости воспитанников для данной группы".

Заполнение табличной части возможно либо вручную по кнопке "Добавить", либо автоматически  по кнопке "Заполнить договорами на зачисление". 

Примечание.
При автоматическом заполнении в табличную часть попадут только те дети, на которых заведен договор о зачислении в данную группу.

При ручном добавлении необходимо сначала выбрать договор о зачислении, при этом остальные колонки заполнятся автоматически на основе данных введенных в договоре.
