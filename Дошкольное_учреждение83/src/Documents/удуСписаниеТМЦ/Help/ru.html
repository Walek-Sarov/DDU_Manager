Документ «Списание ТМЦ»

Документ предназначен для отражения операции списания товарно-материальных ценностей и методических материалов со складов учреждения.

Обязательные реквизиты для заполнения:

Дата - дата документа. По умолчанию ставится текущая дата.

Учреждение - из справочника "Дошкольные учреждения" выбирается ДОУ, со склада которого списываются ТМЦ и методические материалы.

Склад - склад, с которого списываются ТМЦ и методические материалы, выбирается из справочника "Склады (места хранения)". Если для каждого списываемого материала необходимо указывать свой склад, тогда необходимо поставить отметку "Склады в таб. части", в этом случае склад будет указываться в табличной части для каждого материала в отдельности.

Номер - номер документа, присваивается автоматически при записи документа.

Закладка "ТМЦ"

На закладке указываются списываемые товарно-материальные ценности. 

Добавление новых строк в табличную часть происходит по кнопке "Добавить".

В табличной части обязательно указываются наименование ТМЦ (выбирается из справочника "Номенклатура") и количество списываемых единиц, также если ведется учет по инвентарным номерам необходимо указать инвентарный номер.

Закладка "Методические материалы"

На закладке указываются списываемые методические материалы. 

Добавление новых строк в табличную часть происходит по кнопке "Добавить".

В табличной части обязательно указываются наименование материала (выбирается из справочника "Учетные единицы") и количество списываемых единиц, также если ведется учет по инвентарным номерам необходимо указать инвентарный номер.
