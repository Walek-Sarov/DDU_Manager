Документ «Приказ о переводе в другую группу»

Документ располагается в блоке "Приказы о зачислении, переводе, выбытии".

Предназначен для формирования приказа о переводе ребенка из одной группы ДОУ в другую.

Документ может формироваться автоматически из документа "Комплектование групп", при этом в документе появится ссылка на документ основание.

Назначение реквизитов

Номер приказа - номер приказа, присваивается автоматически при записи документа.

Дата приказа - дата создания приказа о переводе.

Учреждение - указывается учреждение, в группу которого зачисляется ребенок. Заполняется автоматически.

Группа (первоначальная) - из справочника "Группы (воспитательные)" выбирается группа, из которой переводятся дети.

Группа для перевода - группа, в которую будут переведны дети, выбирается также из справочника "Группы (воспитательные)".  

Примечание.
После выбора групп рядом появится информация о колличестве человек в каждой из них и о их вместимости. Если при записи документа количество детей в группе окажется больше, чем ее вместимость, программа выдаст сообщение об ошибке и не позволит записать документ. 

Для отключения контроля вместимости группы необходимо установить отметку "Отключить контроль вместимости".

Табличная часть заполняется автоматически. Заполнение возможно либо по договорам на зачисление, либо списком воспитанников указанной группы (первоначальной).

При заполнении составом группы в табличную часть попадут все дети, числящиеся в группе, Необходимо оставить отметку "галочку" только напротив тех, кого следует перевести. 

Примечание.
Если для перевода детей в новую группу не требуется заключение нового договора - необходимо поставить отметку "Разрешить перевод в группу без заключения нового договора".
