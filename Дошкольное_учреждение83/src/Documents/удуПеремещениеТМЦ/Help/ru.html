Документ «Перемещение ТМЦ»

Документ предназначен для отражения операции перемещения товарно-материальных ценностей и методических материалов с одного склада (места хранения) на другой.

Обязательные реквизиты для заполнения:

Дата - дата документа. По умолчанию ставится текущая дата.

Учреждение - из справочника "Дошкольные учреждения" выбирается ДОУ, на складах которого происходит перемещение ТМЦ и методических материалов.

Склад - отправитель - склад, с которого перемещаются ТМЦ или методические материалы, выбирается из справочника "Склады (места хранения)".

Склад - приемник - склад, на который перемещаются ТМЦ или методические материалы, выбирается из справочника "Склады (места хранения)".

Если для каждого перемещаемого материала необходимо указывать свои склады, тогда необходимо поставить отметку "Склады в таб. части", в этом случае склад-отправитель и склад-приемник будут указываться в табличной части для каждого материала в отдельности.

Номер - номер документа, присваивается автоматически при записи документа.

Закладка "ТМЦ"

На закладке указываются перемещаемые товарно-материальные ценности. 

Добавление новых строк в табличную часть происходит по кнопке "Добавить".

В табличной части обязательно указываются наименование ТМЦ (выбирается из справочника "Номенклатура") и количество перемещаемых единиц, также если ведется учет по инвентарным номерам необходимо указать инвентарный номер.

Закладка "Методические материалы"

На закладке указываются перемещаемые методические материалы. 

Добавление новых строк в табличную часть происходит по кнопке "Добавить".

В табличной части обязательно указываются наименование материала (выбирается из справочника "Учетные единицы") и количество перемещаемых единиц, также если ведется учет по инвентарным номерам необходимо указать инвентарный номер.
