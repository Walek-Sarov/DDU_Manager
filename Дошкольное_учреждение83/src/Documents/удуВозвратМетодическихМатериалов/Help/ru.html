Документ «Возврат методических материалов»

Документ предназначен для учета фактов возврата сотрудниками организации ранее полученных методических материалов.

 

В форме данного документа имеются следующие поля:

Дата - дата документа. По умолчанию ставится текущая дата.

Получатель (читатель) - сотрудник учреждения, который выполняет возврат методических материалов.

Причина выдачи - заполняется при необходимости возврата методических материалов, выданных ранее для определнных целей (копириования, реставрации и т.п.).

Дата возврата (план) - срок возврата методического материала.

Учетная единица - возвращаемый методический материал - элемент справочника Учетные единицы.

На руках - количество экземпляров методического материала, находящихся на руках у сотрудника. Заполняется автоматически при выборе методического материала.

Сдать - количество возвращаемых экземпляров.
