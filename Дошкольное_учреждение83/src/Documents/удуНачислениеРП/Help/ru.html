Документ «Начисление родительской платы»

Документ располагается в блоке "Посещаемость учреждения".

Документ предназначен для начисления долга по каждому ребенку.

Начисление формируется за каждый месяц отдельно, для всех детей вместе, либо по отдельности. Можно создать несколько документов Начисление родительской платы за один месяц.

 

Указывается Месяц за который формируется документ. Табличная часть заполняется списком детей либо вручную либо по кнопке "Заполнить по всем".

 

После заполнения документа, его необходимо провести и закрыть.
