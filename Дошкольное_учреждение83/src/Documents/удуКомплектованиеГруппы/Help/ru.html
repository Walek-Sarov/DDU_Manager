Документ «Комплектование групп»

Документ располагается в блоке "Книга учета потребностей".

Документ предназначен для предварительного комплектования групп ДОУ.

Перед созданием нового документа необходимо указать на какой период будут комплектоваться группы. На один и тот же период можно задать только один документ комплектации.

Назначение реквизитов документа

Номер - номер документа. Номер присваивается автоматически при записи документа.

Дата - дата документа.

Учреждение - заполняется автоматически. 

Оплата (по умолчанию) и Содержание договора (по умолчанию)  - реквизиты использующиеся для автоматического формирования договоров на зачисление.

Заполнение табличной части происходит по кнопке "Заполнить табличную часть". Заполняется заявками на зачисление, если в настройках программы стоит отметка об использовании возрастного признака при комплектовании групп, тогда заполнение возможно и составом какой-либо группы ДОУ.

Табличная часть заполняется согласно порядковому номеру (колонка № оч.) ребенка в очереди на зачисление.

После заполнения табличной части можно автоматически сформировать нужные договора на зачисление. для этого необходимо поставить отметку в колонке "ФД" напротив тех детей для которых необходимо сформировать договора. нажать кнопку "Сформировать договора на зачисление". Договор заключается на каждого ребенка в отдельности.

Также можно автоматически сформировать приказы на зачисление. Для этого ставится отметка в колонке "Утв", и нажимается кнопка "Сформировать приказы о формировании группы". Приказы формируются для каждой группы в отдельности. Для этого в табличной части необходимо указать группу для зачисления.

Приказы формируются согласно очереди. Если при формировании приказов произошло переполнение группы (количество человек в группе больше, чем ее вместимость), тогда программа выдаст предупреждение и не сформирует приказы для тех детей, которые выходят за пределы вместимости группы.

После проведения документа зеленым цветом выделятся те строчки с детьми, по которым произошло зачиление в группы ДОУ.
