Документ «Выдача методических материалов»

Документ предназначен для учета фактов выдачи методических материалов на руки. Для выдачи доступны только методические материалы ранее оприходованные с помощью документа Оприходование ТМЦ (имеющие положительный остаток по регистру  ТМЦ на складах).

 

В форме данного документа имеются следующие поля:

Дата - дата документа. По умолчанию ставится текущая дата.

Получатель (читатель) - сотрудник учреждения, которому выдаются методические материалы для работы.

Причина выдачи - заполняется при необходимости указания причины, по которой выдан методический материал.

Дата возврата (план) - срок возврата методического материала.

Учетная единица - выдаваемый методический материал - элемент справочника Учетные единицы.

Количество - количество выдаваемых экземпляров методического материала.
