Загрузка электронной почты позволяет перенести почтовые сообщения из указанного почтового ящика в справочник входящих или внутренних документов.

Для создания входящего или внутреннего документа выберите почтовое сообщение и выполните соответствующую команду меню "Создать на основании".

После создания входящего или внутреннего документа выбранное почтовое сообщение будет автоматически удалено.

Загрузка почтовых сообщений возможна только для учетной записи с установленным флажком "Оставлять копии сообщений на сервере". Регистрация учетных записей выполняется администратором системы.

См. также: Учетные записи электронной почты, Работа с формами.
