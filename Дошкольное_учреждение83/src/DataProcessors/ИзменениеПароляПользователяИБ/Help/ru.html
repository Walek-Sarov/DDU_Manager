Для открытия формы смены пароля следует в разделе "Настройка и администрирование" в панели действий в группе "Сервис" выполнить команду "Изменение пароля пользователя".

Для изменения пароля необходимо ввести новый пароль в оба поля.

Для установки пароля следует нажать кнопку "Сменить пароль". Для отказа от смены - кнопку "Отмена".

Если введенные значения различаются (могут различаться символы, регистр и раскладка клавиатуры), выводится сообщение об ошибке.
