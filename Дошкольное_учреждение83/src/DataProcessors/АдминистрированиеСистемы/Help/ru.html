Выполнение функций по администрирование системы разрешено только пользователю с ролью "Полные права":

  • Удаление помеченных объектов
  • Управление полнотекстовым поиском
  • Управление настройками форм
  • Активные пользователи
  • Консоль регламентных заданий
  • Журнал регистрации
  • Очистка истории работы всех пользователей
