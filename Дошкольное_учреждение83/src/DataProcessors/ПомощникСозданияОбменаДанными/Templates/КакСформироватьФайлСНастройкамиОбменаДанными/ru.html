Файл с настройками обмена данными

Если файл настроек обмена данными утерян, то для формирования нового файла необходимо:

  • перейти в первую информационную базу
  • перейти в форму обменов данными
  • выбрать требуемый обмен данными и открыть его форму
  • в командной панели формы следует выбрать пункт меню: "Параметры обмена данными" –> "Получить настройки обмена данными для второй информационной базы..."
